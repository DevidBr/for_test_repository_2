Как сделать из папки сделать git repository
1. Заходим в папку через git bash или еще как.

2. Выполняем команду $ git init эта команда сообщит гиту, 
что папка теперь репозиторий и за ней надо следить.

3. Проложить связь от локального репозитория до удаленного (который создали на git hub).
Это делается командой $ git remote add origin https://github.com/DevidBr/for_test_repository_2.git
Где:
- git это гит
- remote - команда, упраляющая удаленными репозиториями (например на гитхабе)
- add - субкоманда. 
Вместе git remote add позволяет дать короткое имя (alias) для URL удаленного репозитория.
Обычно, по договоренности, это имя origin.
- https://github.com/DevidBr/for_test_repository_2.git - это просто URL удаленного репозитория на GH.
